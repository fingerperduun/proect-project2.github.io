📖 Что такое HTML?

Основы веб-разработки, которые ты изучишь через игру

🌐 HTML — язык веба

HTML (HyperText Markup Language) — это стандартный язык разметки для создания веб-страниц. Он определяет структуру и содержание страницы с помощью тегов.

  • Создан в 1993 году Тимом Бернерсом-Ли
  • Работает в связке с CSS и JavaScript
  • Интерпретируется всеми браузерами

🏷️ Что такое теги?

Теги — это команды для браузера в угловых скобках. Они говорят браузеру, как отображать контент.

  • Большинство тегов парные: <p>текст</p>
  • Некоторые одиночные: <img>, <br>
  • Теги могут иметь атрибуты: <a href="#">

📦 Категории тегов

Все HTML-теги делятся на группы по назначению:

  • 🏗️ Структура: div, header, footer, section
  • 📝 Текст: p, h1-h6, span, strong
  • 🎨 Медиа: img, video, audio, canvas
  • 📋 Формы: input, button, form, select

🚀 Зачем учить HTML?

HTML — первый шаг в веб-разработке. Без него невозможно создать сайт.

  • Основа для изучения CSS и JavaScript
  • Нужен для любой веб-профессии
  • Понимание структуры сайтов
  • Возможность создавать свои проекты
